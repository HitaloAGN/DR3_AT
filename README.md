# InfnetFood - App de Delivery

InfnetFood é um aplicativo completo de delivery que permite aos usuários navegar por categorias de alimentos, adicionar itens ao carrinho, visualizar restaurantes em um mapa, finalizar pedidos e acompanhar o status das entregas.

## Tecnologias

- React Native
- Expo
- React Navigation (Stack & Bottom Tabs)
- React Native Maps
- Expo Notifications
- Axios
- Animated API

## Como Executar

### Pré-requisitos
- Node.js instalado
- Expo CLI instalado (`npm install -g expo-cli`)
- Expo Go instalado no celular (Android/iOS)

### Instalação

1. Clone o repositório
```bash
git clone https://github.com/HitaloAGN/DR3_AT.git
cd DR3_AT
```

2. Instale as dependências
```bash
npm install
```

3. Inicie o projeto
```bash
npm start
```

4. Escaneie o QR Code com o Expo Go
